## 2. 도메인 인벤토리 (전수)

도메인 패키지 `domain/{term,schema,mapping,pattern,normalize,expand}/`. 6 Aggregate Root + 3 Domain Service.

| 클래스 | 종류 | 마커 | 서브도메인 | 행위/팩토리 | 타 AR 참조 |
|---|---|---|---|---|---|
| `Term` | Aggregate Root | `@AggregateRoot` | CORE | 상태머신(ACTIVE/DRAFT/DEPRECATED) + 팩토리 | — |
| `Synonym` | Aggregate Root | `@AggregateRoot` | CORE | 팩토리 + 행위 | `termId` (ID-only) |
| `SchemaCatalog` | Aggregate Root | `@AggregateRoot` | SUPPORTING | 팩토리 + 행위 | — |
| `CodeValue` | Aggregate Root | `@AggregateRoot` | SUPPORTING | 팩토리 + 행위 | `schemaCatalogId` (ID-only) |
| `SchemaMapping` | Aggregate Root | `@AggregateRoot` | SUPPORTING | 팩토리 + 행위 | `termId`, `schemaCatalogId` (ID-only) |
| `SqlPattern` | Aggregate Root | `@AggregateRoot` | GENERIC | 팩토리 + 매칭 | — |
| `NormalizationService` | Domain Service | (config 등록) | — | 상대시간→TimeRange (stateless) | — |
| `ExpansionService` | Domain Service | (config 등록) | — | 동의어 확장 (stateless) | — |
| `SqlPatternService` | Domain Service | (config 등록) | — | 키워드 매칭 (stateless) | — |

- 도메인 레코드(결과 평탄화): `ColumnMapping`, `CodeValueCandidate`, `SynonymMatch`, `SqlPatternMatch`, `NormalizationResult`, `ExpansionResult`, `TimeRange`. 교차-AR 조인 결과를 도메인 레코드로 평탄화 — 객체 그래프 오염 회피.
- `@DomainService` 빈은 `infrastructure/config/DomainServiceConfig`에서 `@Bean` 등록(도메인은 Spring 무의존).
